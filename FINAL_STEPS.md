
# 🎊 FINAL STEPS - AFAR MKOPO FASTA

---

## 📤 STEP 1: Push to GitHub (1 Minute)
In your terminal, run:
```bash
git push -u origin main
```

When prompted, use your GitHub username and Personal Access Token.

---

## 🌐 STEP 2: Deploy to Vercel (5 Minutes)

1. **Go to Vercel**: Open [vercel.com](https://vercel.com)
2. **Log In**: Click "Continue with GitHub" and log in
3. **Import Project**: Click "Add New..." → "Project", select your `afar-mkopo-fasta` repo
4. **Configure Project**:
   - ✏️ **Root Directory**: Click "Edit" and type `frontend`
   - Leave everything else as default
5. **Deploy**: Click "Deploy"!
6. **Wait 1-2 Minutes**: Your site will build!
7. **Done!**: You'll get a live URL (like `https://afar-mkopo-fasta.vercel.app`)!

---

## 🎉 TEST LOCALLY FIRST (Optional)
Double-click `start.bat` to start both servers!
- Frontend: http://localhost:3000
- Backend: http://localhost:5000
